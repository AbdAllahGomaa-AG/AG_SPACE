"# AG_SPACE"

This workspace contains the Angular project in the `ag-space` folder.

## Project

- Name: `AgSpace`
- Framework: Angular CLI `21.2.10`

## Development Server

From the workspace root:

```bash
cd ag-space
ng serve
```

Open `http://localhost:4200/` in your browser.

## Useful Commands

Run these commands from `ag-space`:

```bash
ng generate component component-name
ng build
ng test
ng e2e
```

## Notes

- Build output is generated in `dist/`.
- Angular CLI command reference: https://angular.dev/tools/cli
